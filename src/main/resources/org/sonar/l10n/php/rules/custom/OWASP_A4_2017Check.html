Noncompliant Code Example

    XML;

    $dom = new DOMDocument();
    $dom->loadXML($xml);

    // Prints Test.
    print $dom->textContent;

Compliant Solution

    XML;

    libxml_disable_entity_loader(true);
    $dom = new DOMDocument();
    $dom->loadXML($xml);

    // Prints Test.
    print $dom->textContent;
