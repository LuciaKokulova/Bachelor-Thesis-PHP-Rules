Noncompliant Code Example

     $password   = substr(sha1(microtime().rand()),0,12);
     $passwordDB = sha1($data[0]['login'].'*'.$password);
     $actPwdLink = sha1(md5(microtime()).rand());

Compliant Solution

     $password   = substr(sha1(microtime().random_int()),0,12);
     $passwordDB = sha1($data[0]['login'].'*'.$password);
     $actPwdLink = sha1(md5(microtime()).random_int());
